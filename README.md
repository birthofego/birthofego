# birthofego

My personal portfolio. Built with Next.js.

## Running it

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Editing content

Everything — projects, copy, links, stack — lives in `src/data/content.ts`. Change it there, save, done.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel
